# AI usage log

I used Cursor (Composer) throughout this project for scaffolding, implementation, and documentation. I ran the API locally with Docker, exercised endpoints with curl, and ran `npm test` before committing. Where AI output looked plausible but failed in practice, I changed or rejected it — notes below.

Added `import 'dotenv/config'` in the db module. **Modified:** lint script stays as `eslint 'src/**/*.ts'` for CI; on Windows I run `npx eslint "src/**/*.ts"` because PowerShell mangles quoted globs. |
